What is Block-Suite?

Block-Suite is a modular desktop application built for Minecraft server security assessments. It provides a collection of tools — each focused on a specific testing scenario — all managed through a clean JavaFX interface with 5 built-in themes, auto-updates, and cross-version support via ViaVersion.

Modules

Each module is a self-contained feature accessible from the main interface.

🔄

Server Cloning Available

Deploy a transparent MITM proxy that mirrors any Minecraft server. Intercept logins, credentials, player data, IPs, and skins in real-time with integrated playit.gg tunneling.
🔧

More coming soon Planned

New assessment modules are in development. Follow the GitHub repository for announcements on upcoming features and release notes.

Platform Highlights

🎨

5 Themes

Dark Blue, Green Suite, Black Green, Purple Suite, Black Purple. Live theme switching with matching icon tinting.

🔃

Auto-Updates

One-click updates from GitHub Releases. Validates checksums and applies updates seamlessly on next launch.

🌐

ViaVersion Support

Connect players on any Minecraft version to any server version. Cross-version protocol translation is built in.

🔒

Modular Architecture

Each tool runs independently. Clean separation between the GUI, proxy engine, plugins, and data layer.

💻

One-Click Setup

No manual configuration. Block-Suite downloads and configures everything automatically — Velocity, plugins, tunnels.

📊

Data Export

Export intercepted data as CSV. Copy individual fields, browse player histories, and inspect geolocation data.
